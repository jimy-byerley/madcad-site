pythonic, extensible, interactive - 3D CAD software

MADCAD is a software and a programming library that allows you to make CAD modeling using finite triangular meshes, with simple and intuitive python scripts. As of any python script, it empowers you with every existing python modules in the world, to compute, dimension or exploit your CAD results.

It behaves too differently than any other CAD system you may have used to make a list here. Checkout an extensive comparison here


The madcad project is splitted:

  • pymadcad — The core code , free and *open-source* as every library should be
  • uimadcad — The user interface , free and *open-source* as well, its development is independent from pymadcad but is occasionnaly guiding it
